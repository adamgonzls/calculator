Tone

SolarCloud UI tries to have a clean tone overall. The styles should match the UI, which represents emergent technology, cleanliness, and education.

Branding

Logo Guidelines

  • Don’t add additional shadows or lighting effects, modify colors, or change the orientation.
  • When positioning the logo beside other elements, please ensure there is sufficient outer padding.

Spacing

Spacing can be applied using margin or padding. There are several available spacing sizes. Both margin and padding share the same predefined scale.

Spacing after <section></section> should normally be 100px.
